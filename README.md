# 🚢 Titanic Survival Prediction - Data Visualization & Machine Learning

## 📌 Project Overview
This project analyzes the famous Titanic dataset to understand the factors that influenced passenger survival.  
It includes data preprocessing, exploratory data analysis (EDA), visualizations, and machine learning model building.

---

## 📊 Dataset
- Source: Kaggle Titanic Dataset
- Features include:
  - PassengerId
  - Pclass
  - Name
  - Sex
  - Age
  - SibSp
  - Parch
  - Fare
  - Embarked
  - Survived (Target Variable)

---

## 📈 Data Visualizations

The project includes the following visualizations:

### 🔹 Survival Count
Shows number of survivors vs non-survivors.

### 🔹 Survival Based on Gender
Bar plot showing survival distribution for male and female passengers.

### 🔹 Passenger Class Distribution
Visualization of passengers in different ticket classes.

### 🔹 Age Distribution
Histogram representing passenger age distribution.

### 🔹 Correlation Heatmap
Heatmap showing correlation between numerical features.
